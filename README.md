## 📥 Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/sandovaldavid/testimonios-page.git
    ```

2. Navigate to the project directory:

    ```bash
    cd testimonios-page
    ```

3. Install dependencies:

    ```bash
    npm install
    ```

## 🔧 Usage

To run the development server:

```bash
npm start
```

To build for production:

```bash
npm run build
```
